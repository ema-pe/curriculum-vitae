# Curriculum vitae

This repository contains the source code of my curriculum vitae, written in
LaTeX using primarily the [CurVe](https://ctan.org/pkg/curve) class.

The Git repository is hosted on both
[GitLab](https://gitlab.com/ema-pe/curriculum-vitae) and
[GitHub](https://github.com/ema-pe/curriculum-vitae), with GitHub serving as a
**read-only mirror** of the GitLab repository.

You can download the latest PDF version of my curriculum vitae
[here](https://gitlab.com/ema-pe/curriculum-vitae/-/jobs/artifacts/master/raw/cv_emanuele_petriglia.pdf?job=build-pdf).
